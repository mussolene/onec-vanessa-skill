# onec-vanessa-skill

Production-ready skill package for agent-assisted 1C testing with:

- Vanessa Automation for UI/BDD flows;
- xUnitFor1C for unit/module tests;
- thin adapters for Codex, Cursor, and Claude;
- cross-platform Python bootstrap, doctor, run, debug, and artifact collection commands;
- thin PowerShell wrappers for Windows teams that prefer `pwsh`;
- git-first packaging for reuse in other repositories.

The repository does not bundle the 1C platform, Vanessa Automation, xUnitFor1C, or `vanessa-runner`. It orchestrates those tools with templates, documented commands, and conservative platform claims.
